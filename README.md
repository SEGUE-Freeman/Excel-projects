# Projet Payroll sous Excel

## Description

Ce projet est un exercice pratique réalisé dans le cadre de mon apprentissage d’Excel, en suivant un tutoriel de FreeCodeCamp.

L’objectif était de construire un système simple de gestion de la paie permettant d’organiser les informations des employés et d’automatiser certains calculs liés aux salaires à l’aide des fonctionnalités d’Excel.

## Fonctionnalités

- Gestion des informations des employés
- Calcul automatique des salaires
- Organisation des données de paie
- Utilisation de formules Excel pour automatiser les calculs

## Outils utilisés

- Microsoft Excel

## Compétences développées

- Manipulation de données sous Excel
- Utilisation de formules et fonctions Excel
- Organisation et structuration de données
- Automatisation de calculs simples
- Conception de feuilles de calcul

## Objectif du projet

Mettre en pratique les concepts fondamentaux d’Excel à travers la réalisation d’un mini système de gestion de la paie.

## Source

Projet réalisé à partir d’un tutoriel d’apprentissage proposé par FreeCodeCamp.
